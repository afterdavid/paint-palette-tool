# Target Manufacturers

This file records the initial catalog targets for Paint Palette Tool.

## Priority set

### Native / highly portable
- Behr
- Glidden

### Universal
- Sherwin-Williams
- Benjamin Moore

### Usually available through competitor libraries
- Dunn-Edwards
- Valspar

## Notes

The point of this list is not just color completeness. It is field usefulness.

These brands were selected because they are practical to take to real counters and mural workflows.

The product should preserve both:
- the original manufacturer identity
- the practical library context in which muralists encounter the colors
