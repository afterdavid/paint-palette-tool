# Manufacturer Catalogs

This directory will hold normalized full-catalog files for target paint manufacturers.

Current catalog files:
- `paint-store-accessible.json` — combined best-current source across target brands
- `behr.json` — official Behr ColorSmart payload, deduped by color id
- `glidden.json` — official Glidden sitemap/detail-page import
- `ppg.json` — official PPG sitemap/detail-page import
- `dunn-edwards.json` — official Dunn-Edwards digital swatch import
- `valspar.json` — official Valspar browse-colors import
- `sherwin-williams.json` — partial official crawl
- `benjamin-moore.json` — partial official crawl

Derived or source-scoped companion files may also appear when an official source publishes
curated subsets or alternate libraries that should stay separate from the main brand catalog
(for example, professional/designer palette subsets).

Each catalog should use the canonical schema fields and preserve provenance.

The combined catalog is generated by:

```bash
python3 scripts/build_accessible_catalog.py
```
