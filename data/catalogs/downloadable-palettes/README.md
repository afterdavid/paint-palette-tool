# Downloadable palette outputs

These files are **source-derived outputs from official downloadable palette/library assets**.

They are intentionally separated from `data/catalogs/*.json`, which is reserved for canonical manufacturer catalog imports.

Why separate them?
- downloadable palette assets are excellent official swatch-library sources
- but they are not automatically verified as complete live manufacturer catalogs
- keeping them separate preserves provenance and avoids overstating completeness
